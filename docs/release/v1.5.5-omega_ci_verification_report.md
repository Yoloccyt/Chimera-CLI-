## 1. 修复内容

### 问题: Docker 镜像体积验证失败

**症状**: release.yml `Docker Image` job 的 `Verify image size < 100MB` 步骤失败:
```
Error response from daemon: No such image: ghcr.io/yoloccyt/chimera-cli:v1.5.4-omega
```

**根因**: `docker buildx build --push` 仅推送镜像到 registry,不会在本地 Docker 存储中保留镜像。直接 `docker image inspect` 找不到镜像。

**修复**: 在 `Verify image size < 100MB` 步骤中添加 `docker pull "$IMAGE"` 命令,先拉取镜像到本地,再执行 `docker image inspect`。

### 问题: Docker 构建因 Rust 版本过旧失败

**症状**: `docker build` 报 `feature 'edition2024' is required` 错误(exit code 101)。

**根因**: `hashbrown-0.17.1` 等依赖需要 `edition2024` 特性(Cargo 1.85.0 稳定化),但 Dockerfile 使用 `rust:1.82-slim`(Cargo 1.82.0 不支持)。

**修复**: Dockerfile builder 基础镜像从 `rust:1.82-slim` 升级至 `rust:1.85-slim`。
